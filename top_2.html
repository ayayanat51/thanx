PERFUMJOURNAL

The number of perfumes
is proportional to
the number of memories

02

I always wear perfume when I go out, find it very relaxing to do my favorite. I also choose one perfume for each trip or special memory because smell is strongly associated with memory. This allows me to vividly recall memories when I smell the perfume. I believe perfume is the best memory storage tool, even better than photographs.
